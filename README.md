# Wings Coaching - Full Stack Setup

## Run Locally

```powershell
pip install -r requirements.txt
python app.py
```

Open: `http://127.0.0.1:5000`

## What This Includes

- Premium animated frontend (`index.html`, `styles.css`, `script.js`)
- Python Flask backend (`app.py`)
- Lead capture API: `POST /api/enquiry`
- Local lead storage in `data/leads.jsonl`
- Auto WhatsApp message link on submit to `+91 95597 52997`
